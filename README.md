# SnackTrack

SnackTrack is a conference-first snack tracker for people who want to stay calorie-aware during events without logging every bite.

## Features

- Quick snack logging for event food
- Conference and game night presets
- Macro summary with an easy pie chart
- Editable macro goals
- Shareable summary text for demo use
- Empty-state start with no default food history

## Tech Stack

- React
- Vite
- LocalStorage for demo persistence

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
